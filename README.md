# DataToRag Marketplace

Claude Code plugin marketplace for [DataToRag](https://github.com/DataToRag).

## Plugins

| Plugin | Description |
|--------|-------------|
| [claude-pg-mem](https://github.com/DataToRag/claude-pg-mem) | Postgres-native persistent memory for Claude Code |
| [datatorag](https://github.com/DataToRag/mcp-gateway) | Connect Claude to Google Workspace, Atlassian, and more via MCP |

## Install

```
claude install DataToRag/marketplace
```
